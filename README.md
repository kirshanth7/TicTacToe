# TicTacToe - Human vs Human (Java)

A console-based Tic-Tac-Toe game built in Java where two players play against each other.

## How to Play
- Player 1: Bob (X)
- Player 2: Yash (O)
- Enter row and col (0-2) when prompted
- First to get 3 in a row wins!

## Concepts Used
- Static 2D arrays
- OOP — multiple classes (TicTacToe, human)
- Input validation with do-while loop
- Win detection — row, column & diagonal

## How to Run
1. Clone the repo
2. Open in Eclipse
3. Run `launchgame.java`
